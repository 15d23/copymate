##CopyMate

Copy Anywhere, Paste once.


####Demo

	⌘+C -> ⌃⇧C -> ⌃⇧C -> ⌃⇧2 -> ⌃⇧C -> ⌘V  
	
	Copy -> Append Copy -> Append Copy -> Change separator to space -> Append Copy -> Paste
